Saute

An iOS meal planning assistant generating weekly plans from users' pantries, taste profiles, dietary constraints, and schedules. Implemented receipt scanning for pantry tracking using vision OCR, tokenization, and fuzzy matching with MongoDB. Built recipe ranking with ingredient expiry urgency, ingredient reuse as a DAG for multi-day meal plans, and Google Calendar API syncing for automated meal scheduling.
