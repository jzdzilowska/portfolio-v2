Drowsiness Detection

A real-time ML driver drowsiness detection system achieving 98% accuracy on balanced datasets. Built using TensorFlow, VGG16, and CNN architectures. Automated video-frame extraction, facial-region detection using OpenCV and Haar cascades, and labeling pipeline. Applied L1/L2 regularization and dropout to reduce overfitting. Demonstrated system effectiveness in accident prevention on live webcam input simulating drowsiness conditions.
