Runwave

A real-time CFO dashboard for SME founders to monitor cash flow, manage invoices, and receive AI-generated financial insights. Aggregated bank transactions via Plaid into Postgres and developed a concurrent FastAPI service generating cash-flow forecasts and financial health insights using structured SQL analytics and Gemini. Implemented automated Gmail invoice ingestion with PDF parsing and GPT-4o extraction, Stripe payment processing, and a context-aware Gemini chat agent operating over live financial data.
